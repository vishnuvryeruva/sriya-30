# Video goes here

Drop ONE video file in this folder named **montage.mp4** (or update the
filename in config.js → video.mp4Src).

IMPORTANT — GitHub size limit:
- Each file must be UNDER 100 MB or the push will be rejected.
- Aim for under ~50 MB so it loads fast on phones at the party.
- If your video is bigger, either:
  (a) compress it (e.g. handbrake.fr, or `ffmpeg -i in.mp4 -vf scale=-2:720 -crf 28 montage.mp4`), or
  (b) use the YouTube "Unlisted" option instead (see config.js).
