# Photos go here

Drop your image files into this folder, then list their filenames in
`config.js` under the `gallery` array.

**Tips for a fast, beautiful page:**
- Name them simply: `photo-01.jpg`, `photo-02.jpg`, … (matches the defaults in config.js)
- JPG or WebP work best. Aim for ~1600px on the long edge.
- Keep each file under ~500 KB so the page loads quickly from a QR code on mobile.
- Any filename in `config.js` that has no matching file just shows a soft
  "Photo coming soon" placeholder — nothing breaks.
